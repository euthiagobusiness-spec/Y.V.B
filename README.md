# Yasmin Vitoria Beauty

Landing page responsiva em React, Vite e Tailwind CSS para a marca Yasmin Vitoria Beauty.

## Desenvolvimento

```bash
npm install
npm run dev -- --port 5173
```

URL local:

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```

O build de producao e gerado em `dist/`.

## Deploy

Configuracao recomendada para plataformas como Vercel, Netlify ou similares:

- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`
